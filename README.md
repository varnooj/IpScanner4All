# 🚀 Fast Python IP Scanner

یک اسکریپت سبک، سریع و بهینه با پایتون برای اسکن کردن لیستی از آی‌پی‌ها و شبکه‌ها (CIDR). این ابزار با استفاده از پردازش موازی (Multithreading)، می‌تواند میلیون‌ها آی‌پی را در زمان کوتاهی بررسی کرده و آی‌پی‌های فعال (دارای پینگ) را شناسایی کند.

## ✨ ویژگی‌ها (Features)

* **پشتیبانی از رنج آی‌پی:** علاوه بر آی‌پی‌های تکی، قابلیت استخراج و اسکن فرمت‌های CIDR (مانند `192.168.1.0/24` یا `/26` و غیره) را دارد.
* **سرعت بالا (Concurrent Scanning):** استفاده از ۵۰ رِشته (Thread) همزمان برای پینگ گرفتن که سرعت اسکن را به شدت افزایش می‌دهد.
* **ذخیره در لحظه (Real-time Saving):** نتایج منتظر پایان اسکن نمی‌مانند و در همان لحظه در هارد دیسک نوشته می‌شوند تا در صورت قطعی برق یا بسته شدن ترمینال، هیچ دیتایی از دست نرود.
* **توقف اضطراری هوشمند (Graceful Exit):** پشتیبانی کامل از `Ctrl+C`. در صورت توقف دستی، برنامه در کسری از ثانیه متوقف شده و خروجی‌ها تا همان لحظه حفظ می‌شوند.
* **تعیین تعداد پینگ:** امکان تنظیم تعداد درخواست‌های پینگ (Ping Count) برای هر آی‌پی در زمان اجرا.
* **بدون وابستگی (No External Dependencies):** این پروژه کاملاً با کتابخانه‌های استاندارد و داخلی پایتون نوشته شده و نیازی به `pip install` ماژول‌های جانبی ندارد.
* **Cross-Platform:** به طور خودکار سیستم‌عامل (ویندوز، لینوکس، مک) را تشخیص داده و پارامترهای پینگ را متناسب با آن تنظیم می‌کند.

## 📥 نصب و راه‌اندازی (Installation)

ابتدا مخزن را کلون کنید و وارد پوشه پروژه شوید:

```bash
git clone https://github.com/varnooj/IpScanner4All.git
cd IpScanner4All

```
🚀 نحوه استفاده (Usage)
برای اجرای اسکریپت، به یک فایل متنی نیاز دارید که شامل لیست آی‌پی‌ها یا رنج‌های شبکه باشد (مثلاً ips.txt). هر آی‌پی یا رنج باید در یک خط جداگانه نوشته شود.

نمونه فایل ورودی (ips.txt):


1.1.1.1
8.8.8.8
13.113.196.64/26
15.207.13.128/25
اجرای دستور خط فرمان:
می‌توانید برنامه را به دو صورت اجرا کنید:

۱. حالت پیش‌فرض (فقط تعیین فایل ورودی):
در این حالت آی‌پی‌های فعال در فایلی به نام active_ips.txt ذخیره می‌شوند.

```bash
python scanner.py ips.txt
```

۲. حالت دلخواه (تعیین فایل ورودی و خروجی):
می‌توانید نام فایل خروجی را در آرگومان دوم وارد کنید.

```bash
python scanner.py ips.txt result.txt
پس از اجرا، برنامه از شما می‌پرسد که مایلید هر آی‌پی چند بار پینگ شود (عدد پیش‌فرض ۴ است):
```

🛑 توقف اسکن
در هر مرحله از اسکن، در صورتی که نیاز به توقف برنامه داشتید، کافیست از میانبر Ctrl + C در کیبورد استفاده کنید. برنامه بلافاصله اسکن را متوقف کرده و به شما پیام می‌دهد که چند آی‌پی تا این لحظه در فایل خروجی با موفقیت ذخیره شده‌اند.
